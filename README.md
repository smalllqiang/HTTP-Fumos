# HTTP Fumos

## 简介

模仿**HTTP Cats**而来的项目, 用fumo来给状态码增加一点乐趣吧

## 使用方法

> `https://http-fumos.top/[status_code]`

如果需要一个后缀名, 在最后加 `.jpg` 即可.
