# HTTP Fumos

## 简介

模仿HTTP Cats而来的项目, 用fumo来给状态码增加一点乐趣吧  

## 使用方法

直接使用 `https://http-fumos.top/{status_code}` 即可得到返回的图像  
如果需要后缀名的话, 使用 `.jpg`  

## 为该项目贡献

目前需要高质量 切合含义的fumo图  
可以在 issue 中提交图片, 也可加入我们的QQ交流群: `1067568439`  
对图像的要求:  
- jpeg格式
- 不含有额外的元数据
- 大小适度
- 不侵犯版权
